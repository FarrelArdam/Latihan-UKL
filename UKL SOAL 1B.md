Fungsi:

Program ini digunakan untuk menentukan apakah sebuah bilangan adalah ganjil atau genap.

Cara Kerja:

Input bilangan:
Pengguna memasukkan satu angka melalui Scanner.

Pengecekan dengan operator modulus (%):

Jika bilangan % 2 == 0, maka bilangan genap.

Jika tidak, maka bilangan ganjil.

Tampilkan hasil:
Program menampilkan pesan “bilangan ganjil” atau “bilangan genap”.
